# ClearPage 跨浏览器测试报告

## 测试环境
- 操作系统：Windows 11
- 浏览器：Chrome 120、Edge 120、Firefox 120（目标版本）

## 测试维度与用例
- 响应式布局：在 480/600/720/960px 宽度下侧栏与主区比例正确，主区滚动正常
- 视觉层次：标题、正文、辅助文本层级清晰，间距一致
- 组件一致性：按钮/输入/下拉与卡片样式统一，hover/active/disabled 状态正确
- 动效与无障碍：`focus-visible` 焦点环显示，`prefers-reduced-motion` 生效；键盘导航顺序合理
- 兼容性：扩展弹窗加载正常，语言切换与性能页功能正常

## 结果摘要
- Chrome：通过（弹窗宽高、滚动、样式与交互一致）
- Edge：通过（Chromium 内核一致）
- Firefox：样式与交互通过；MV3 异步能力与权限需按 Firefox 扩展规范单独验证

## 已修复/注意项
- 侧栏文本强制单行，避免换行导致排版断裂
- 焦点可见性统一至 `focus-visible:ring`，满足键盘可访问性
- 减少动效模式下关闭过渡，提升可达性
